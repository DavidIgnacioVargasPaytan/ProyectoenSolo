# God of War - El Portal Definitivo

## URL del Proyecto Hospedado
El proyecto se encuentra desplegado y funcional en el siguiente enlace: https://screackvp.pythonanywhere.com/

## Descripción del Proyecto
Este portal web es una enciclopedia interactiva dedicada a la franquicia de videojuegos "God of War". Permite a los usuarios explorar la evolución de Kratos a través de las sagas griega y nórdica, consultar información detallada sobre personajes, armas y bosses, y escuchar la banda sonora. Además, cuenta con un sistema de **Registro e Inicio de Sesión** conectado a una base de datos SQLite, permitiendo a los usuarios tener un perfil personalizado dentro de la plataforma ("Screack Productions").

## Tecnologías Utilizadas
Este proyecto ha sido desarrollado implementando una arquitectura Full-Stack con las siguientes tecnologías:

* **HTML5:** Estructura semántica de las páginas.
* **CSS3:** Estilos, diseño responsivo, animaciones y tema oscuro inmersivo.
* **JavaScript:** Interactividad del menú, modales, galería y validaciones asíncronas (Fetch API).
* **Python (Flask):** Lógica del servidor (Backend), enrutamiento y manejo de sesiones.
* **SQL (SQLite):** Base de datos relacional para la gestión de usuarios, noticias y jefes.
